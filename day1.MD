### 笔记
- 在本地情况下，master是主分支，还有dev分支
- git branch
- git checkout dev
- git commit
- 合并冲突：git pull;git add .;git commit -m hiahiahia;git push
- 如何退出修改页面：esc;wq
- 如何修改：i可移动i；按esc,再按dd可删除
- 如何push:git pull;vim (文件）.MD;git add .;git commit -m 随便打打；git push
- 如果无法push，则需重新pull
